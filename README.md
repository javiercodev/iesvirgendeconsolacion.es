# Ejecutar y desplegar

Esto contiene todo lo que necesitas para ejecutar la aplicación de forma local.


## Ejecutar localmente

**Requisitos previos:**  Node.js


1. Instalar dependencias:
   `npm install`

2. Ejecutar la aplicación:
   `npm run dev`
